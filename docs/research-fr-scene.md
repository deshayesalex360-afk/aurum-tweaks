## Tweaks consensus FR (intégrés dans Aurum Tweaks)

1. Désactiver Game Bar / Game DVR
2. Game Mode + HAGS
3. Plan Ultimate Performance
4. Désactiver hibernation
5. Fullscreen Optimizations off
6. Télémétrie (DiagTrack disabled)
7. Cortana / Web Search off
8. Widgets Win11 off
9. Visual effects réduits
10. NetworkThrottlingIndex = 0xFFFFFFFF
11. SystemResponsiveness = 10
12. TCP TcpAckFrequency + TCPNoDelay
13. MenuShowDelay = 0
14. Superfetch SSD off
15. Power Throttling off
16. Storage Sense off
17. Debloat AppX (Bing*, Xbox*, OfficeHub, etc.)
18. Restore point auto
19. Reset Winsock + flush DNS
